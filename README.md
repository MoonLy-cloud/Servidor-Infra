# 🚀 DogoCorp Infrastructure & Labs

Repositorio centralizado de mi infraestructura personal y entorno de prácticas. Administrado en un servidor Ubuntu con Docker y Nginx Proxy Manager.

## 🛠️ Proyectos Incluidos

### 1. 📂 Herramientas Base
Configuración principal de servicios de infraestructura:
* Nginx Proxy Manager: Gestión de subdominios y certificados SSL (Let's Encrypt).
* Portainer: Administración visual de contenedores.
* Nextcloud: Nube privada de almacenamiento de archivos.

### 2. 🐍 Python Flask Web
* Ubicación: `/dogocorp-flask`
* Página web personal de prueba corriendo sobre Flask y desplegada con Docker.

### 3. ☕ Spring Boot Lab (Próximamente)
* Ubicación: `/SpringBoot-TEST`
* Entorno dedicado para mis prácticas universitarias de backend con Java/Spring.
* Configurado en el puerto -9090- para evitar conflictos.

### 4. 🎮 Gaming Servers
* Terraria & Minecraft: Servidores dedicados para pruebas de latencia y persistencia de datos.
